# Telegram Log Monitor Bot

Бот для мониторинга логов в Telegram с гибкой настройкой через регулярные выражения. Отслеживает важные события в лог-файлах и отправляет уведомления подписчикам.

## Возможности

- **Мониторинг в реальном времени** - отслеживает новые записи в лог-файлах
- **Гибкая фильтрация** - настройка регулярных выражений для поиска нужных событий
- **Telegram уведомления** - мгновенные оповещения подписчиков
- **Цветовая маркировка** - визуальное разделение типов событий
- **Автоархивация** - автоматическое архивирование старых логов
- **Безопасность** - работа через официальное Telegram API
- **Производительность** - читает только новые записи, оптимизированные regex
- **Отказоустойчивость** - обработка ошибок и автоматическое восстановление

## Требования

- Python 3.8 или выше
- Telegram аккаунт
- Токен бота от @BotFather

## Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/yourusername/telegram-log-monitor-bot.git
cd telegram-log-monitor-bot
```

### 2. Создание виртуального окружения

```bash
# Linux/Mac
python -m venv venv
source venv/bin/activate

# Windows
python -m venv venv
venv\Scripts\activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Настройка токена бота

Создайте файл `.env` в корневой директории:

```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота_здесь
```

Как получить токен:
1. Найдите в Telegram [@BotFather](https://t.me/botfather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям
4. Скопируйте полученный токен

### 5. Запуск

```bash
python bot.py
```

## Настройка

### Конфигурация паттернов (config.py)

Отредактируйте файл `config.py` для настройки отслеживаемых событий:

```python
LOG_PATTERNS = {
    'errors': {
        'pattern': r'ERROR|CRITICAL|FATAL',
        'description': 'Критические ошибки',
        'color': '🔴',
        'enabled': True
    },
    'database': {
        'pattern': r'DB_ERROR|CONNECTION_FAILED|TIMEOUT',
        'description': 'Ошибки базы данных',
        'color': '🔵',
        'enabled': True
    },
    # Добавьте свои паттерны
    'custom': {
        'pattern': r'YOUR_REGEX_HERE',
        'description': 'Описание',
        'color': '🟢',
        'enabled': True
    }
}
```

### Дополнительные настройки

```python
# Интервал проверки логов (секунды)
CHECK_INTERVAL = 60

# Максимальный размер файла логов (байты)
MAX_LOG_SIZE = 1024 * 1024  # 1 МБ

# Путь к файлу логов
LOG_FILE = "logs.log"
```

## 📱 Использование

### Команды бота

|     Команда    |           Описание           |
|----------------|------------------------------|
| `/start`       | Приветственное сообщение     |
| `/help`        | Показать справку             |
| `/status`      | Проверить статус мониторинга |
| `/subscribe`   | Подписаться на уведомления   |
| `/unsubscribe` | Отписаться от уведомлений    |

### Пример работы

1. **Подписка на уведомления**
   ```
   /subscribe
   ```

2. **Проверка статуса**
   ```
   /status
   ```
   Ответ:
   ```
   📊 Статус мониторинга
   
   🟢 Бот активен
   📁 Файл логов: logs.log
   ⏱ Интервал проверки: 60 сек
   📝 Подписка: ✅ активна
   👥 Всего подписчиков: 1
   
   *Отслеживаемые паттерны:*
     🔴 Критические ошибки
     🟡 Предупреждения
     🔵 Ошибки базы данных
   ```

3. **Получение уведомлений**
   ```
   📨 Новые события в логах
   🕐 2024-01-15 10:30:25
   📊 Всего событий: 3
   
   🔴 *Критические ошибки* (2)
   ```
