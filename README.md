# Telegram Log Monitor Bot

Бот для мониторинга логов в Telegram с гибкой настройкой через регулярные выражения. Отслеживает важные события в лог-файлах и отправляет уведомления подписчикам.

## Возможности

- 📊 **Мониторинг в реальном времени** - отслеживает новые записи в лог-файлах
- 🎯 **Гибкая фильтрация** - настройка регулярных выражений для поиска нужных событий
- 📱 **Telegram уведомления** - мгновенные оповещения подписчиков
- 🎨 **Цветовая маркировка** - визуальное разделение типов событий
- 📁 **Автоархивация** - автоматическое архивирование старых логов
- 🔒 **Безопасность** - работа через официальное Telegram API
- ⚡ **Производительность** - читает только новые записи, оптимизированные regex
- 🛡 **Отказоустойчивость** - обработка ошибок и автоматическое восстановление

## Использование

Команды бота:

Команда	        Описание
/start	        Приветственное сообщение
/help	        Показать справку
/status	        Проверить статус мониторинга
/subscribe	    Подписаться на уведомления
/unsubscribe	Отписаться от уведомлений

Настройка цветов
Доступные эмодзи для маркировки:

🔴 Красный - критические ошибки

🟡 Желтый - предупреждения

🔵 Синий - информационные сообщения

🟣 Фиолетовый - отладка

🟢 Зеленый - успешные операции

⚪ Серый - системные сообщения
